# EduFinance (Spring Boot Edition)

> A simplified version of **EduFinance**, a personal/educational gamified finance management and learning system, built as the final project for our technical degree at **Instituto Federal Sul-rio-grandense (IFSul)**.

This repository contains a leaner version of the original EduFinance project, focused on demonstrating the core back-end architecture with **Java + Spring Boot**, **PostgreSQL**, and **JPA/Hibernate**, paired with a lightweight **HTML/CSS/JS** front end.

---

## 📌 About the Project

EduFinance was developed to help users learn finances in a simple and accessible way. This Spring Boot edition strips the original project down to its essentials, making it easier to read, study, and extend — ideal as a reference implementation or learning resource.

This is a companion/educational version of the main EduFinance project, co-developed with [Ygor Bitencourt](https://github.com/YgorBDev).

---

## ✨ Features

- User registration and authentication
- Income and expense tracking
- Category-based transaction organization
- Monthly/period financial summaries
- RESTful API built with Spring Boot
- Data persistence with PostgreSQL via JPA/Hibernate
- Simple, responsive front end (HTML, CSS, JS)
- The HTML and CSS were made with AI, using Antigravity IDE.
